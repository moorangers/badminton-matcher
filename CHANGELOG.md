# Changelog

## v0.3.0 (2026-05-16)

- แยก flow เปลี่ยนตัวผู้เล่น ออกจาก ลบผู้เล่น ชัดเจน
- เปลี่ยนตัวแล้วผู้เล่นเดิมจะไปพัก (ไม่ถูกลบออกจากรายชื่อ)
- ปรับการนับสถิติรายคนให้เกิดตอนจบแมตช์เท่านั้น
- เพิ่มการจัดการผู้เล่นผ่าน Modal (แก้ชื่อ / ลบผู้เล่น)
- เพิ่ม validation ตอนแก้ชื่อ (ห้ามว่าง, ห้ามชื่อซ้ำ, Enter เพื่อบันทึก, Esc เพื่อปิด)
- เพิ่มตัวนับแมตช์รวมที่เล่นไปแล้ว และแสดงในแถบสรุปใต้กระดานแมตช์
- ปรับปรุงเอกสาร README ให้สอดคล้องกับพฤติกรรมล่าสุดของระบบ
